# CSLT_Nhom1_quanLyVeXemPhim
Chương trình Quản lý vé xem phim là ứng dụng console được viết bằng ngôn ngữ C#.   Mục tiêu của chương trình là mô phỏng hệ thống quản lý vé cơ bản trong rạp phim, cho phép: - Hiển thị sơ đồ ghế - Đặt và hủy vé - Tìm kiếm, sắp xếp và chỉnh sửa thông tin vé khách hàng - Thống kê và xem lịch sử đặt vé
